# Clothing Store WordPress Theme

This directory contains a basic WordPress theme for an eCommerce clothing store.

## Installation

1. Copy the `wordpress_theme_clothing` folder into your WordPress installation under `wp-content/themes`.
2. Log in to the WordPress admin panel and activate **Clothing Store** from the Appearance > Themes menu.

## Features

- Simple responsive layout
- Product listing on the main page
- Support for featured images and navigation menu

This theme is provided as a minimal example and may require additional customization for production use.
